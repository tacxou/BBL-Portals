BBL Portals

# 1.1.1
- Fixed custom portals not showing in JEI when used in KubeJS

# 1.1.0
- Added JEI support for custom portals, shows size, type, frame and ignition method
- All portals will be shown in JEI, unless using showInJEI(false) on the portal builder and kubejs builder
- Disabled default portals added by this mod unless in dev
- Enabled curseforge and modrinth auto updating

# 1.0.0

- Initial release

# Changes and fixed from Portal JS and Custom Portal API
- Color tint now works correctly
- Portals now longer broken when hit by flowing water
- No longer crashes when flint and steel is used in world
- Portal color now correct when inside the portal
- Now longer crashes on startup
- Added portal block translation
- Added Sodium support for portal rendering
- Changed particles to be a lot more natural and tint the same colors and the portal

# New Features
- Jade support to show the dimension that you will be teleported to when looking at the portal
- .showParticles(true/false) optional defaults to true, allows you to disable the portal particles if you want
- .portalTexture("default"/"nether"/"molten") optional defaults to default, allows you to change the portal texture to either the default one, the nether portal one or a molten one. All textures respect the portal color tint
